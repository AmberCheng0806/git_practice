# git_practice
## GIT練習筆記

GIT => command line 版本控制軟體


儲存庫=> 本地端儲存庫(Commit) 遠端儲存庫(Push)
一定需要先在本地儲存庫建立暫存，後續才能異動到遠端儲存庫


MS => M$
遠端儲存庫(儲存程式碼的雲端硬碟)
GitHub => 做為全世界最大的開發者平台(開源)
GitLab=> 中小企業最愛用的平台 (開源 自行託管)
Bitbucket=> 大型公司 有錢企業愛用平台 (一條龍的服務)



Git指令:
.gitignore 建立不監聽的檔案
download:單純純下載(變成壓縮檔) 不支援程式碼追蹤
clone:將遠端儲存庫同步到本地端去建立程式碼追蹤(GIT)
fork:將不屬於該組織的專案拷貝一份到自己的repo(建立自己的儲存庫)
commit:將此次的版本異動記錄下來並且儲存到本地端
push:將該次commit紀錄同時備份到遠端
